Start local server.

## Installation

```bash
$ yarn global add sar
# or
$ npm global install sar
```

## Usage

Simple usage.

```bash
$ sar [dir] # Start local server in dir
```

SPA mode.

```bash
$ sar [dir] --spa
```


## Help

```bash
Usage:
  $ start [dir]

Commands:
  [dir]  Start static local server.

For more info, run any command with the `--help` flag:
  $ start --help

Options:
  -p, --port     port
  -s, --spa      spa flag
  -h, --help     Display this message
  -v, --version  Display version number
```
